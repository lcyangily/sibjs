一、属性参数

名称 可选值 描述
target null, string, dom node 必传，ID
border true, false 表格外是否带边框，不用可设置false，或者写样式覆盖
width 'auto', number 表格宽度，默认auto为自适应。宽度改变会重新计算里面每个列宽。
height 'auto', number 高度自适应或者指定高度
minHeight 200, number height为'auto'时, 此值生效,最低高度,当没有数据或者数据很少时很有用。
frozenColumns null, array 固定列，该列固定在坐左边，当有横向滚动条时，这些列不会滚动，方便查看。支持多级表头，所以是二维数组。
columns null, array 列属性，二维数组。支持多级表头。
striped true, false 表格偶数行奇数行不同颜色
method 'post', 'get' 表格Ajax查询数据提交方式
idField null, array 表格主键，可以是一个列(string)或者联合主键(array)，这样更新或选择可以通过主键来更新一行，而不用通过行坐标。
url null, string 表格数据请求地址
loadMsg 'Processing, please wait ...', string 加载数据提示信息，可自定义，实现国际化
noDataMsg string 没有查询到记录的提示信息，自定义
pagination false, true 是否显示分页
rownumbers false, true 是否显示行记录是第几条
singleSelect true, false 是否单选，每次只能选中一行
pageNumber 1, number 当前页码
pageSize 10, number 每页条数
pageToolbar null, string, DOM node 分页工具条，可自定义操作。显示在分页条上
showPageNumber 10, number 分页条显示的页码数量
pageDisplayMsg '共{total}条/{pages}页', string 分页条上显示的信息，{total}为总数据量，{pages}为总页数
queryParams {}, object 数据查询条件
paramType null, string('json') 查询提交的参数类型，如果为json，会自动转成json类型传递
sortName null, string 排序的列名
sortOrder 'asc', 'desc' 升序还是降序
remoteSort true, false 是否为远程后台排序，也支持前台当前页数据的排序(需要在列属性指定列数据类型，默认string)。
contentType null, string http传递时的 contentType,默认不用改写
data null, array 数组，表格显示的数据。
viewMode 'default', string 插件。还有'detail'模式(可实现Sub Grid)，可自行开发插件
resultsLocation 'row', string 查询结果集数组的位置。如返回{a : 'aaa', b : {c : '', d :[...]}}配置'b.d',当前搜索结果为d对应的数组，可灵活配置结果集。
totalLocation 'total', string 查询结果集总数字段的位置。如返回{a : 'aaa', b : {c : 100, d :[...]}}配置'b.c',当前搜索结果总数，可灵活配置。
remotePageCurrentLabel 'current', string 查询的页码名，Server 端接受数据的属性名。后台无需接受指定参数，前台可灵活定制。
remotePageSizeLabel 'pageSize', string 每页显示数名，Server 端接受数据的属性名。后台无需接受指定参数，前台可灵活定制。
remoteSortNameLabel 'sortName', string 排序字段名，Server 端接受数据的属性名。后台无需接受指定参数，前台可灵活定制。
remoteSortOrderLabel 'sortOrder', string 排序类型名，Server 端接受数据的属性名。后台无需接受指定参数，前台可灵活定制。
onLoadSuccess null, function 加载数据成功的回调函数
onLoadError null, function 加载数据失败的回调函数
onClickRow null, function(rowIndex, rowData){} 点击一行的处理函数
onDblClickRow null, function(rowIndex, rowData){} 双击一行的处理函数
onSortColumn null, function(sort, order){} 排序的回调函数
onSelect null, function(rowIndex, rowData){} 选择一行的回调函数
onUnselect null, function(rowIndex, rowData){} 取消选择一行的回调函数
onResize null, function(){} 尺寸改变回调函数

二、列属性参数

名称 可选值 描述
sortable false, true 列是否可排序
visible true, false 列是否可见
field null, string 列对应数据的属性名
title null, string 列显示的标题
dataType 'string', 'number' 列数据类型，排序时候很有用，
formatter function(){}, 'function' 列数据显示格式化function(colData, rowData, $td){}
入参包括三个参数：colData:列数据，rowData:行数据, $td当前表格格子的DOM对象
headerFormatter function(){}, 'function' 列数据显示格式化headerFormatter(col, $th){}
入参包括三个参数：col:列定义, $th当前表格头格子的DOM对象

三、方法

名称 入参 描述
resize 重新调整尺寸
reload param 重新加载表格数据
getSelected msg:string 得到第一个被选中的行记录
getSelectedIndex 获的第一个选择的坐标,没有返回 -1
getRowIndex 通过行记录获得坐标,需要设置idField, 没有idField或找不到值返回-1
getSelections 得到所有选中的记录
clearSelections 清除所有被选中的
selectRow index 通过记录数组下标选择一行
unselectRow index 通过记录数组下标取消选择一行
selectRecord idValues 通过id值得到一行记录
unselectRecord idValues 取消选择
isSelected row 判断行记录是否被选中
getRow index 通过记录数组下获取行数据
getRows 获取所有行数据
refresh 刷新表格
updateRow index, row 更新一行
updateRowByIdField idFields, row 更新一行

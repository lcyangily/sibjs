一、属性参数

名称 可选值 描述
target null, string, dom node 必传，ID
appendTo null, dom node 搜索结果弹出面板父级元素，模式在body下
autoFocus false, true 自动选择第一个搜索结果
delay 300, number 输入框值改变延迟查询时间,越小越灵敏,但性能也越差
minLength 1, number 查询的最小长度值(小于次长度不查询)
source null, object, array, string, function(request, response){} 搜索的数据源,支持对象，数组，ajax url, 函数
sourceParam {}, json 查询的静态参数
method 'get', 'post' ajax请求 http 提交类型,get还是post
zIndex 1000, number 弹出面板的z-index值
position object 弹出层的位置,来自jqueryui position.js插件
listItemFormatter null, function(item, $itemBody){} 查询结果自定义每条显示格式或样式，个性化定制显示数据。入参为行记录和行的父节点
enableAutoFill false, true 是否在失去焦点时，自动填充选择的记录
noResultsMessage string 没有查询到记录的提示信息，自定义
selectMode false, true 选择模式，失去焦点会判断输入的值可在数据源中，如果不在则自动清空
resultsLocation nullstring 查询结果集数组的位置。如返回{a : 'aaa', b : {c : '', d :[...]}}配置'b.d',当前搜索结果为d对应的数组，可灵活配置结果集。
labelName 'label', string 结果元素显示的属性名
valueName 'value', string 结果元素的值
hotSource null, object, array, string, function(request, response){} 推荐面板的数据源，格式同source
hotWidth 'auto', number 推荐面板宽度
hotMaxWidth 320, number 推荐面板最大宽度
hotMode 'table', string, object 推荐面板插件,如果为string会自动系在plugins对应的插件文件,目前支持'table', 'tab'
hotOptions {}, object 推荐面板的参数，见具体插件，随具体插件而定
change null, function 输入值改变回调函数
close null, function 显示面板关闭回调
select null, function 选择一条记录回调

二、方法

名称 可选值 描述
showResultsPanel 显示查询结果集面板
showHotPanel 显示推荐面板
showMessage msg:string 显示提示信息
hideHotPanel 隐藏推荐面板
hideResultsPanel 隐藏结果集面板
hide 隐藏面板
labelValue 获取搜索框中的值
value null, value 获取或设置值
search msg:string 查询指定字符串
setOptions object 设置属性

一、属性参数

名称 可选值 描述
target null, string, dom node 必传，ID
threme 'default', string 主题样式default, point 对应具体样式为sib-stepbar-default,sib-stepbar-point,可自定义
allColors ['orange','blue','green','red','pink','gray'], array 可供选择的颜色,如果有更多，可添加
color 'blue', string 当前的颜色，allColors中的一个值
stepsSelector 'li', string 初始化时，原页面中节点，jquery selector
nodes null, array 初始化节点数，此处有值则忽略stepsSelector选择的页面节点
labelName 'label', string 节点元素显示的属性名
valueName 'value', string 节点元素的值属性名
border true, false 是否有边框
width 'auto', number 步骤条宽度
minWidth 300, number 步骤条最小宽度
stepWidth 'auto', number 步骤条每个节点的最小宽度
stepMinWidth 50, number 步骤条每个节点的最小宽度
act 0, number 当前第几步, 从1开始。 1,2,3... 0未开始,nodes.length + 1 结束
onStepClick nullfunction 点击节点回调

二、方法

名称 可选值 描述
resize 重新计算调整尺寸
reload nodes : Array 重新加载显示步骤条，入参为节点数组
setCurrentIndex number 通过节点索引index设置当前步骤
setCurrentValue value 通过节点value设置当前执行哪一步
getCurrent 得到当前节点的数据
getCurrentIndex 得到当前节点的坐标
getNodes 得到所有节点
setStyle color : string 设置颜色，入参为allColors 所有颜色中的一个
setThreme threme : string 设置主题
prev 上一步
next 下一步
complete 完成

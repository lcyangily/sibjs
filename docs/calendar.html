一、属性参数

名称 可选值 描述
inlineNode null, dom node, jquery element 不为空为内联模式，嵌入html页面中，为嵌入的父级节点
trigger null, dom node, jquery element 非内联模式，触发显示的节点
triggerType 'click', string 非内联模式，触发显示的时间名。click、mouseover等
lang 'en', string 国际化语言，懒加载，对应i18n文件夹下的资源文件
date null, string, moment对象 初始化日期，moment时间对象或满足格式（format指定格式）的字符串
format 'YYYY-MM-DD', string 日期格式化
range null, [string, string] 或者 [moment, moment] 日历可选择的时间范围，第一个是开始时间第二个是结束时间
startDay 0, number[0-6] 每周的第一天，默认是从周日开始，可自由选择
hideOnSelect true, false 选择日期后是否自动隐藏时间面板
output null, dom node, jquery element 日期输出到哪里，默认跟trigger同一个元素
showOtherMonths true, false 在日期当月是否显示上一月和下一月日期填满空白
selectOtherMonths true, false 在日期当月显示的上一月和下一月日期是否可点击
showWeek false, true 是否显示周
show null,function 显示日历的回调函数
hide null, function 隐藏日历的回调函数
selectDate null, function 选择日期的回调函数
selectMonth null, function 选择月份的回调函数
selectYear null, function 选择年份的回调函数

二、方法

名称 可选值 描述
show 显示日历
hide 隐藏日历
output val : string 输出日期到配置的output节点中,日历的日期或者输入的日期
enable 使日期控件可用
disable 使日期控件无用

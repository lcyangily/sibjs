一、属性参数

名称 可选值 描述
target null, string, dom node 必传，导航菜单的id
navSelector 'li a', string target元素下的选择器，导航菜单所有子元素
navSelectedClass 'selected', string 选中样式
enableReverse true, false 反向控制,滚动页面时，自动高亮选中对应的导航链接
interval 400, number 页面滚动延迟计算时间
threshold 0, number 显示导航的临界高度,页面scrollTop小于这个高度导航自动隐藏
duration 800, number 页面滚动时间
easing 'swing', string 滚动动画效果
scrollbegin null, function 滚动开始回调
scrollend nullfunction 滚动结束回调
